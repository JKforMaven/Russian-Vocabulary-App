# Учи́м ру́сский! — Russian Vocabulary App

A simple web app for learning the most common Russian words using flashcards and quizzes.

**Live app:** https://jkformaven.github.io/Russian-Vocabulary-App/

## Features

- **Learn mode** — flip through flashcards, mark words as known
- **Quiz mode** — multiple-choice quiz to test yourself
- **Progress tracking** — tracks your quiz score and known words over time
- **5 difficulty tiers** — from basic words (и, в, не) to more advanced vocabulary
- **500 words** — Russian with English and Czech translations
- **Works on any device** — phone, tablet, laptop

## How to use

Just open the live link above. Your progress is saved in your browser automatically — no account needed.

## Tech stack

Plain HTML, CSS, and JavaScript. No frameworks, no build tools, no server needed.

## Files

| File | What it does |
|------|-------------|
| `index.html` | The main page structure |
| `style.css` | All the visual styling |
| `app.js` | App logic — flashcards, quiz, navigation, progress |
| `words.js` | The word list — 500 Russian words with English and Czech translations |

## Running locally

Open `index.html` in any browser. That's it!
